Welk formaat wordt er eigenlijk gebruikt om het Date object?

Het formaat getFullYear(), getMonth(), getDate(), getHours(), getMinutes(), getSeconds(), getMilliseconds()

Wat gebeurt er als eenzelfde object meermaals voorkomt bij het opbouwen van de JSON tekst?

Dubbele vermeldingen van hetzelfde object in JSON worden vervangen door referenties naar het oorspronkelijke object om circulaire verwijzingen te voorkomen.
