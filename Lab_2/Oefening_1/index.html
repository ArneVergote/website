Return value van de confirm functie: Als de gebruiker op 'OK' klikt, is de return value true. Als de gebruiker op 'Cancel' klikt, is de return value false. Return value van de prompt functie: Als de gebruiker een tekst invoert en op 'OK' klikt, is de return value de ingevoerde tekst. Als de gebruiker op 'Cancel' klikt, is de return value null.
